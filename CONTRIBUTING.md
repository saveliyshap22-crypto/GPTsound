# Как помочь GPTsound

Спасибо за интерес к проекту. Для изменений создайте отдельную ветку, опишите пользовательский эффект и убедитесь, что проходят:

```bash
gradle testDebugUnitTest lintDebug assembleDebug
```

## Темы

Новый дизайн можно добавить без изменения Kotlin-кода:

1. Скопируйте `themes/aurora.json`.
2. Измените цвета и параметры согласно `themes/theme.schema.json`.
3. Проверьте JSON через импорт из буфера в приложении.
4. Добавьте файл в `themes/` отдельным pull request.

## SoundCloud

- Не добавляйте client ID, client secret, access token или cookie в репозиторий.
- Используйте только документированные OAuth 2.1 и Public API endpoints.
- Не добавляйте скачивание треков, обход ограничений или блокировку рекламы SoundCloud.
- Сохраняйте ссылку на автора и исходный трек.
